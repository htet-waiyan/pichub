Data Presentation and UI/UX flow of Feed and User Profile
==========================================================

Entities where data are retrieved to be populated on Feed

- photos from threads that the user created/subscribed
- photos with uploader, datetime, thread name

Entities where data are retireved to be populated on Profile

- Personal info to be edited
- number of followings and followers
